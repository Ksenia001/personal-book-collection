# Диаграммы компонентов и развертывания

Данная диаграмма объединяет два представления:
*   **Диаграмма компонентов** показывает, из каких логических блоков состоит приложение.
*   **Диаграмма развертывания** показывает, на каких физических узлах (устройствах) эти блоки работают.

![Диаграмма компонентов и развертывания](https://github.com/Ksenia001/personal-book-collection/blob/main/Diagrams/Images/Components.png?raw=true)

# Описание узлов развертывания

| Узел | Описание |
|:---|:---|
| **User's Computer** | Физическое устройство пользователя (персональный компьютер или ноутбук), на котором запускается приложение. |
| **Web Browser** | Программная среда (например, Google Chrome, Mozilla Firefox), работающая на компьютере пользователя. Является средой выполнения для всего приложения. |

# Описание артефактов

| Артефакт | Описание |
|:---|:---|
| **Application (HTML/CSS/JS)** | Физический набор файлов, составляющих клиентское приложение. Этот артефакт загружается и исполняется веб-браузером. |
| **Local Storage** | Встроенное в браузер хранилище типа "ключ-значение". Используется в качестве локальной базы данных для хранения коллекции книг пользователя. |

# Описание компонентов

| Компонент | Описание |
|:---|:---|
| **User Interface** | Логический компонент, отвечающий за все визуальное представление и взаимодействие с пользователем. Реализуется классами `MainPage` и `BookDialog`. |
| **Application Logic** | Компонент, содержащий основную бизнес-логику приложения. Принимает события от UI и управляет данными. Реализуется классом `LibraryController`. |
| **Data Management** | Компонент, отвечающий за управление данными: их хранение, извлечение и модификацию. Реализуется классами `Library` и `Book` и использует артефакт `Local Storage`. |
