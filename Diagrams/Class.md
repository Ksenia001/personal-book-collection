# Диаграмма классов

Диаграмма классов описывает статическую структуру системы, показывая классы, их атрибуты, методы и отношения между ними. Эта диаграмма отражает архитектуру приложения, разделенную на слои представления (UI), логики (Controller) и данных (Model).

![Диаграмма классов](https://github.com/Ksenia001/personal-book-collection/blob/main/Diagrams/Images/ClassDiagram.png?raw=true)

# Описание классов

| Класс | Описание |
|:---|:---|
| **Book** | Основной класс данных, представляющий книгу в коллекции. Хранит всю информацию о книге: название, автор, год, жанр, оценку и комментарий. |
| **Library** | Класс-хранилище, отвечающий за управление всей коллекцией объектов `Book`. Предоставляет методы для добавления, обновления, удаления и поиска книг. |
| **LibraryController** | Класс-посредник (контроллер), который обрабатывает логику приложения. Он принимает запросы от пользовательского интерфейса и вызывает соответствующие методы у класса `Library`. |
| **MainPage** | Класс, представляющий главный экран пользовательского интерфейса. Отвечает за отображение списка книг и обработку основных действий пользователя (нажатия кнопок добавления, редактирования, удаления, поиска). |
| **BookDialog** | Класс, представляющий диалоговое окно для добавления или редактирования книги. Отвечает за сбор данных от пользователя и их передачу контроллеру. |
